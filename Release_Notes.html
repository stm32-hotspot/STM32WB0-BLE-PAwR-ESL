Purpose

This package contains two specific applications to demonstrate Bluetooth Periodic Advertising with Response (PAwR) on STM32WB09 MCU. See README.md file for more details.

This example is based on the STM32CubeWB0_v1.0.0 package. For further information about these components, see official package.

BLE_ESL and BLE_ESL_AP examples are being delivered as-is, not necessarily supported by ST.

Update History

Main Changes

First release, aligned to official STM32CubeWB0 v1.0.0 package, of Bluetooth PAwR and ESL applications for STM32WB09 MCU

Components

Refer to Release Notes of offical STM32CubeWB0 v1.0.0 package for details of Components (Drivers, Cortex-M CMSIS, STM32 CMSIS, STMWB0x_HAL_Driver, STM32WB0x_Nucleo, Middlewares, Utilities)
